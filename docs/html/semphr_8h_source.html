344 /*
345  * xSemaphoreAltTake() is an alternative version of xSemaphoreTake().
346  *
347  * The source code that implements the alternative (Alt) API is much
348  * simpler because it executes everything from within a critical section.
349  * This is the approach taken by many other RTOSes, but FreeRTOS.org has the
350  * preferred fully featured API too. The fully featured API has more
351  * complex code that takes longer to execute, but makes much less use of
352  * critical sections. Therefore the alternative API sacrifices interrupt
353  * responsiveness to gain execution speed, whereas the fully featured API
354  * sacrifices execution speed to ensure better interrupt responsiveness.
355  */
