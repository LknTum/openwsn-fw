14 Program the IoT-lab_M3 in Windows
15 ---------------------------------
16 
17 1. Download `openocd` Windows binary package from http://openocd.org/
18 1. Extract to `C:\openocd` (or similar)
19 1. Add `C:\openocd\bin` (32-bit Windows) or `C:\openocd\bin-x64` (64-bit Windows) to your Windows `PATH` environment variable.
20 1. Rename the openocd executable from `openocd-xxx-x.x.x.exe` to `openocd.exe`. The OpenWSN bootloader Python script expects `openocd.exe` to be found in the `PATH`. You can test all is OK by typing `openocd.exe` in a Windows command prompt.
21 1. Positions the switch on the IoT-lab_M3 mote in the position *away* from the USB connector.
22 1. Connect the IoT-lab_M3 mote to the USB port of your Windows computer.
23 1. Download Zadig from http://zadig.akeo.ie/. This will use to install the `libusb-win32` drivers openocd requires.
24 1. Open the Zadig executable. Click `Options` > `List All Devices`.
25 1. Two entries corresponding to the same `IoT-lab_M3` node should appear:
26  1. `M3 (Interface 0)`
27  1. `M3 (Interface 1)`
28 1. For interfacing with openocd, one of the two ports must be associated with the OpenOCD `usb-win32_ft2232` driver
29 available in the OpenOCD installation (see [tutorial](https://github.com/hikob/openlab/wiki/Installation-Notes-for-Windows-Users)). We will use `M3 (Interface 0)`.
30 1. Select `M3 (Interface 0)` from the drop down list in Zadig. Select `libusb-win32 (xxx)` driver and click on the button to replace/install the driver for this interface. Once this is done successfully, openocd in Windows can interact with IoT-lab_M3 node and flash it.
31 1. To reprogram the mote, use the usual OpenWSN build scripts. Navigate to the `openwsn-fw/` directory and type `scons board=iot-lab_M3 toolchain=armgcc bootload=0`
32 
33 
34 NOTE: openocd does not support flashing using typical `/dev/ttyUSB*` or `COMX` interfaces. The Python script will flash the first IoT-lab_M3 node connected to the PC, independent from the `bootload=` argument value.

37 Program the IoT-lab_M3 in Linux
38 -------------------------------
39 
40 1. install openocd by issuing 'sudo apt-get install openocd'
41 2. To flash, use the same OpenWSN workflow as in the Windows instructions above.
