#define BSP_TIMER_PERIOD   32768

This program shows the use of the "bsp_timer" bsp module.

Since the bsp modules for different platforms have the same declaration, you can use this project with any platform.

Load this program onto your board, and start running. It will enable the BSP timer. The BSP timer is periodic, of period BSP_TIMER_PERIOD ticks. Each time it elapses:

  • the frame debugpin toggles
  • the error LED toggles
