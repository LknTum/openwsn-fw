This is a program which shows how to use the "openserial" driver module.

Since the driver modules for different platforms have the same declaration, you can use this project with any platform.

This application allows you to test that the openserial driver it working fine. Once your board is running this application, use the serialTesterCli Python application (part of the openwsn-sw repo) to issue serial echo commands, making sure all is well.
