owerror_t(* crypto_engine::aes_ccms_dec) (uint8_t *a, uint8_t len_a, uint8_t *m, uint8_t *len_m, uint8_t *nonce, uint8_t l, uint8_t key[16], uint8_t len_mac)

CCM* inverse transformation (i.e. decryption + tag verification).

Parameters
[in]aPointer to the authentication only data.
[in]len_aLength of authentication only data.
[in,out]mPointer to the data that is both authenticated and encrypted. Overwritten by plaintext.
[in,out]len_mLength of data that is both authenticated and encrypted, including the trailing authentication tag. On return it is reduced for len_mac octets to account for the removed authentication tag.
[in]nonceBuffer containing nonce (max 13 octets).
[in]lCCM parameter L that allows selection of different nonce length which is (15 - L). For example, l = 2 selects 13-octet long nonce which is used for IEEE 802.15.4 security.
[in]keyBuffer containing the secret key (16 octets).
[in]len_macLength of the authentication tag.

owerror_t(* crypto_engine::aes_ccms_enc) (uint8_t *a, uint8_t len_a, uint8_t *m, uint8_t *len_m, uint8_t *nonce, uint8_t l, uint8_t key[16], uint8_t len_mac)

CCM* forward transformation (i.e. encryption + authentication).

Parameters
[in]aPointer to the authentication only data.
[in]len_aLength of authentication only data.
[in,out]mPointer to the data that is both authenticated and encrypted. Overwritten by ciphertext and the trailing authentication tag. Buffer must hold len_m + len_mac.
[in,out]len_mLength of data that is both authenticated and encrypted. Accounts for the added authentication tag of len_mac octets on return.
[in]nonceBuffer containing nonce (max 13 octets).
[in]lCCM parameter L that allows selection of different nonce length which is (15 - L). For example, l = 2 selects 13-octet long nonce which is used for IEEE 802.15.4 security.
[in]keyBuffer containing the secret key (16 octets).
[in]len_macLength of the authentication tag.
