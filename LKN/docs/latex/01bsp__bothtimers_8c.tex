This program shows the use of both the \char`\"{}bsp\+\_\+timer\char`\"{} and \char`\"{}radiotimer\char`\"{} bsp modules. 

Since the bsp modules for different platforms have the same declaration, you can use this project with any platform.

Load this program onto your board, and start running. It will enable the B\+SP timer and radiotimer\+:
\begin{DoxyItemize}
\item the B\+SP timer is periodic, of period B\+S\+P\+\_\+\+T\+I\+M\+E\+R\+\_\+\+P\+E\+R\+I\+OD ticks. Each time it elapses\+:
\begin{DoxyItemize}
\item the frame debugpin toggles
\item the sync L\+ED toggles
\end{DoxyItemize}
\item the radiotimer\+:
\begin{DoxyItemize}
\item overflows every R\+A\+D\+I\+O\+T\+I\+M\+E\+R\+\_\+\+O\+V\+E\+R\+F\+L\+O\+W\+\_\+\+P\+E\+R\+I\+OD
\item during that period, R\+A\+D\+I\+O\+T\+I\+M\+E\+R\+\_\+\+N\+U\+M\+\_\+\+C\+O\+M\+P\+A\+R\+ES compare events fire, every R\+A\+D\+I\+O\+T\+I\+M\+E\+R\+\_\+\+C\+O\+M\+P\+A\+R\+E\+\_\+\+P\+E\+R\+I\+OD ticks.
\end{DoxyItemize}
\end{DoxyItemize}
