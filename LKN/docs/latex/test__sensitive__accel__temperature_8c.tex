This is a standalone test program for the sensitive accelometer and temperature sensor of the G\+I\+N\+A2.\+2b/c board. 

Measurement are typically done simultaneously because they both output analog voltages.

Download the program to a G\+I\+NA board, run it, and break after a few seconds at the line indicated in the code. Use the Watch Window to see the value of \textquotesingle{}sensitive\+\_\+accel\+\_\+x\textquotesingle{}, \textquotesingle{}...\+\_\+y\textquotesingle{}, \textquotesingle{}...\+\_\+z1\textquotesingle{}, \textquotesingle{}...\+\_\+z2\textquotesingle{} and \textquotesingle{}temperature\textquotesingle{}.

The analog input are\+:
\begin{DoxyItemize}
\item P6.\+1\+: Y axis
\item P6.\+2\+: X axis
\item P6.\+5\+: Z axis, after first order filter (abbreviated Z1)
\item P6.\+6\+: Z axis, after third order filter (abbreviated Z3)
\item P6.\+7\+: temperature
\end{DoxyItemize}

The digital outputs to the accelerometer are\+:
\begin{DoxyItemize}
\item P6.\+3\+: pull high enables the accelerometer and analog filter
\item P6.\+4\+: full scale selection (0\+: +/-\/2g full-\/scale; 1\+: +/-\/6g full-\/scale)
\end{DoxyItemize}

The debug pins are\+:
\begin{DoxyItemize}
\item P1.\+1 toggles at every measurement
\end{DoxyItemize}

Speed\+:
\begin{DoxyItemize}
\item one measurement alone takes $\sim$35s
\item one measurement followed by the conversion to real temperature takes $\sim$65us
\end{DoxyItemize}
