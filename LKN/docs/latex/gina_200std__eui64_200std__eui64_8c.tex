This is a standalone program to program the E\+U\+I-\/64 address of a the G\+I\+NA and basestation G\+I\+NA boards. 

Set the variables \textquotesingle{}board\+\_\+type\textquotesingle{}, \textquotesingle{}board\+\_\+version\textquotesingle{} and \textquotesingle{}board\+\_\+id\textquotesingle{}, compile and run the program.

It will store the corresponding E\+U\+I64 into the information memory space of the M\+S\+P430, which does not get erased, ever. The available memory ranges are\+:
\begin{DoxyItemize}
\item 0x10ee-\/0x10f5 \mbox{[}8B\mbox{]} to store the E\+U\+I64
\item 0x10c4-\/0x10d9 \mbox{[}22B\mbox{]} reserved for future use.
\end{DoxyItemize}

The E\+U\+I-\/64 naming convention is\+: 14-\/15-\/92-\/xx yy-\/yy-\/zz-\/zz where\+:
\begin{DoxyItemize}
\item xx is the type of board\+: . 0x09 for G\+I\+NA (\textquotesingle{}9\textquotesingle{} looks like a \textquotesingle{}g\textquotesingle{}) . 0x0b for basestation G\+I\+NA
\item yy-\/yy is the version . 0x022b for a G\+I\+NA 2.\+2b . 0x022c for a G\+I\+NA 2.\+2c . 0x0301 for a G\+I\+NA basestation M\+S\+P1 (\textquotesingle{}3\textquotesingle{} looks like a rotated \textquotesingle{}m\textquotesingle{})
\item zz-\/zz is the unique identifier of the board, in hex, big endian (the one printed on the sticker).
\end{DoxyItemize}
