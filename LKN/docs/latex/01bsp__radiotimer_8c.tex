This program shows the use of the \char`\"{}radiotimer\char`\"{} bsp module. 

Since the bsp modules for different platforms have the same declaration, you can use this project with any platform.

This application can be used to test the radiotimer. It arms the timer to overflow every R\+A\+D\+I\+O\+T\+I\+M\+E\+R\+\_\+\+O\+V\+E\+R\+F\+L\+O\+W\+\_\+\+P\+E\+R\+I\+OD. During that period, it schedules R\+A\+D\+I\+O\+T\+I\+M\+E\+R\+\_\+\+N\+U\+M\+\_\+\+C\+O\+M\+P\+A\+R\+ES events, one every R\+A\+D\+I\+O\+T\+I\+M\+E\+R\+\_\+\+C\+O\+M\+P\+A\+R\+E\+\_\+\+P\+E\+R\+I\+OD.

Each time the radiotimer overflows\+:
\begin{DoxyItemize}
\item the frame debugpin toggles
\item the error led toggles Each time a radiotimer compare event happens\+:
\item the slot debugpin toggles
\item the radio led toggles
\end{DoxyItemize}
